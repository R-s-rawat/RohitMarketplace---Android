C:\ANDROID-DEVELOPMENT\
├── platform-tools\         ← ADB tools (adb.exe, etc.)
├── cmdline-tools\
│   └── latest\             ← Contains bin/, lib/, etc.
├── build-tools\            ← Will be installed later by sdkmanager
├── platforms\              ← Will be installed later by sdkmanager

System Variables:
Variable	    Value
ANDROID_HOME	C:\ANDROID-DEVELOPMENT

Add to System Path:
    C:\ANDROID-DEVELOPMENT\platform-tools
    C:\ANDROID-DEVELOPMENT\cmdline-tools\latest\bin

Then, install (other dependencies) (not to be added to env) - using:
sdkmanager.bat --sdk_root=%ANDROID_HOME% "platforms;android-34" "build-tools;34.0.0"
